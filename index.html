  • UX/UI

    Mi objetivo principal es crear experiencias de usuario excepcionales para aplicaciones, sitios web y productos digitales.

    Investigo, diseño y desarrollo interfaces que sean intuitivas, además de buscar el equilibrio entre la funcionalidad y la estética, para así brindar a los usuarios una experiencia única y satisfactoria.

  • Branding

    Como diseñadora de branding, creo una identidad visual única y coherente para marcas, conectando emocionalmente con su audiencia objetivo.

    Trabajo junto a los clientes para entender su visión y objetivos, y utilizo técnicas de diseño para crear elementos visuales impactantes. Mi objetivo es fortalecer la imagen de la marca y generar reconocimiento y confianza en el mercado.

  • Diseño Web / Frontend

    Como diseñadora de frontend, mi objetivo es crear interfaces atractivas y funcionales para sitios web y aplicaciones.

    Utilizo HTML, CSS y JavaScript para convertir diseños en código, asegurando una experiencia óptima en diferentes dispositivos superando así expectativas y ofrecer una gran experiencia de usuario.
